# PRD Step 2 — 레시피 생성 + 재료 편집 + 필터링

## 개요
Step 1에서 인식한 재료를 바탕으로 AI(Qwen 3.6 Plus)가 맞춤 레시피를 생성합니다. 재료 편집과 식단 필터 기능을 추가합니다.

## 기술 스택
- Step 1 스택 전부 유지
- **추가 AI 모델**: qwen/qwen3.6-plus:free (텍스트 생성)

## 핵심 기능

### 1. 레시피 생성 페이지
- 사이드바에 [🍳 레시피 생성] 메뉴 추가
- 현재 인식된 재료 목록 표시
- [레시피 생성] 버튼 클릭 시 Qwen 모델 호출
- 모델 폴백: qwen3.6-plus → gpt-oss-20b → gemma-4-31b-it → llama-3.3-70b

### 2. 식단 필터
- 채식, 비건, 글루텐 프리, 저염식, 저칼로리, 고단백 옵션
- 선택한 필터를 프롬프트에 반영

### 3. 레시피 카드 표시
각 레시피 카드에 포함:
- **요리명**
- **난이도** (쉬움 / 보통 / 어려움)
- **조리시간** (예: 20분)
- **칼로리** (예: 350kcal)
- **필요 재료** 목록
- **조리 순서** (3~5단계)

### 4. 레시피 저장 버튼
- 각 레시피 카드에 [💾 저장] 버튼 배치
- 로그인 상태일 때만 저장 가능 (Step 3에서 구현)

## 완료 기준
- [ ] 사이드바에서 [레시피 생성] 메뉴로 전환 가능
- [ ] 재료 기반 레시피 3개 생성
- [ ] 요리명/난이도/조리시간/칼로리 표시
- [ ] 식단 필터 적용 시 조건을 반영한 레시피 출력
