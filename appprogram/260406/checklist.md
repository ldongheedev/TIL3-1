## 2단계: 핵심 로직 및 UI 구현

| # | 확인 항목 | 완료 |
|---|---------|:----:|
| 1 | `index.html`, `style.css`, `js/main.js` 파일이 존재한다 | ✅ |
| 2 | `js/quiz.js`, `js/timer.js`, `js/score.js` 파일이 존재한다 | ✅ |
| 3 | `js/leaderboard.js`, `js/share.js` 파일이 존재한다 | ✅ |
| 4 | 6개 화면(시작/난이도/퀴즈/피드백/결과/리더보드)이 HTML에 정의되어 있다 | ✅ |

| 10 | 퀴즈 화면 상단에 카테고리·난이도·문제번호·현재점수·콤보가 표시된다 | ✅ |
| 11 | 타이머 프로그레스 바가 문제 표시와 동시에 작동을 시작한다 | ✅ |
| 12 | 난이도별 제한시간이 올바르게 설정된다 (easy=30s / normal=20s / hard=15s) | ✅ |
| 13 | 보기 선택 즉시 타이머가 정지된다 | ✅ |
| 14 | 타이머 보너스 점수가 남은 시간 비율 구간에 따라 계산된다 | ✅ |
| 15 | 콤보 보너스가 3/5/10연속 기준으로 지급되고 팝업이 표시된다 | ✅ |
| 16 | 오답 또는 시간 초과 시 콤보 카운터가 0으로 초기화된다 | ✅ |
| 17 | 결과 화면에 최종 점수·등급·정답률·최고콤보가 모두 표시된다 | ✅ |
| 18 | Noto Sans KR 폰트가 적용되고 미니멀 레이아웃이 구현되어 있다 | ✅ |
| 19 | 모바일 360px 기준 레이아웃이 깨지지 않는다 | ✅ |
| 20 | 키보드 1~4로 보기 선택, Enter로 다음 문제 진행이 작동한다 | ✅ |
