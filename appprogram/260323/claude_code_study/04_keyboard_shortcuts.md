# Claude Code 단축키 전체 정리

---

## 모드 전환

| 단축키 | 동작 |
|--------|------|
| `Shift+Tab` | Normal → **Plan Mode** → Auto-Accept 순환 |
| `Alt+T` (Windows) / `Option+T` (Mac) | Extended Thinking 토글 |
| `Alt+P` (Windows) / `Option+P` (Mac) | 모델 전환 |

---

## 일반 제어

| 단축키 | 동작 |
|--------|------|
| `Ctrl+C` | 현재 입력·생성 취소 |
| `Ctrl+D` | Claude Code 종료 (EOF) |
| `Ctrl+L` | 터미널 화면 지우기 (대화 히스토리 유지) |
| `Ctrl+O` | 상세 출력(Verbose) 토글 — 도구 사용 과정 확인 |
| `Ctrl+R` | 명령 히스토리 역방향 검색 |
| `Ctrl+G` | 현재 입력·응답을 외부 에디터에서 열기 |
| `Ctrl+V` / `Cmd+V` | 클립보드 이미지 붙여넣기 |
| `Ctrl+B` | 백그라운드 실행 (tmux 사용자는 두 번) |
| `Ctrl+T` | 태스크 목록 표시 토글 |
| `Ctrl+F` | 모든 백그라운드 에이전트 중단 (3초 내 두 번 눌러 확인) |
| `Esc` `Esc` | 이전 시점으로 Rewind |
| `↑` / `↓` | 명령 히스토리 탐색 |
| `←` / `→` | 다이얼로그 탭 전환 |

---

## 텍스트 편집

| 단축키 | 동작 |
|--------|------|
| `Ctrl+K` | 커서 이후 줄 끝까지 삭제 (붙여넣기 버퍼에 저장) |
| `Ctrl+U` | 현재 줄 전체 삭제 (붙여넣기 버퍼에 저장) |
| `Ctrl+Y` | 삭제된 텍스트 붙여넣기 |
| `Alt+Y` | 붙여넣기 히스토리 순환 (`Ctrl+Y` 이후 사용) |
| `Alt+B` | 커서를 한 단어 뒤로 이동 |
| `Alt+F` | 커서를 한 단어 앞으로 이동 |

---

## 멀티라인 입력

| 방법 | 단축키 | 지원 터미널 |
|------|--------|------------|
| 백슬래시 + 엔터 | `\` + `Enter` | 모든 터미널 |
| macOS 기본 | `Option+Enter` | macOS 기본 터미널 |
| 일반적 | `Shift+Enter` | iTerm2, WezTerm, Ghostty, Kitty |
| 제어 문자 | `Ctrl+J` | 라인피드 문자 |

---

## 입력 접두사

| 접두사 | 동작 |
|--------|------|
| `/` | 슬래시 명령어 또는 스킬 실행 |
| `!` | Bash 명령어 직접 실행 |
| `@` | 파일 경로 자동완성 |
| (Space 길게) | 음성 받아쓰기 (Push-to-talk, 설정 필요) |

## Vim 모드 단축키 (`/vim` 으로 활성화)

### 모드 전환
| 키 | 동작 |
|----|------|
| `Esc` | NORMAL 모드 진입 |
| `i` `I` `a` `A` | INSERT 모드 (각각 위치 다름) |
| `o` `O` | 아래/위에 새 줄 추가 후 INSERT |

### NORMAL 모드 이동
| 키 | 동작 |
|----|------|
| `h` `j` `k` `l` | 좌/하/상/우 이동 |
| `w` | 다음 단어 시작 |
| `b` | 이전 단어 시작 |
| `0` | 줄 시작 |
| `$` | 줄 끝 |
| `gg` | 입력 시작 |
| `G` | 입력 끝 |

### NORMAL 모드 편집
| 키 | 동작 |
|----|------|
| `x` | 현재 문자 삭제 |
| `dd` | 현재 줄 삭제 |
| `yy` | 현재 줄 복사 |
| `p` / `P` | 커서 뒤/앞에 붙여넣기 |
| `>>` | 들여쓰기 |
| `J` | 아래 줄 합치기 |
| `.` | 마지막 편집 반복 |

## 커스텀 단축키 설정

`/keybindings` 명령어로 `~/.claude/keybindings.json` 파일을 열어 직접 설정 가능.

```json
{
  "submit": "enter",
  "newline": "shift+enter",
  "voice": "space"
}
```
