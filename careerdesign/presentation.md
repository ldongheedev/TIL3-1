2. 시각자료, 왜 중요할까?
시각자료는 정보를 더 쉽게 이해하고, 더 오래 기억하게 하며, 더 강하게 설득하는 도구입니다.

시각자료를 활용한 효과
이해를 돕는다
 - 영상, 그래프, 도표 등을 활요ㅇ
2. 	기억에 오래 남게 한다
 - 발표자의 말과 시각자료가 더해졌을 때 효과적으로 전달
3. 흥미를 유발한다
 - 스티브 잡스의 맥북 에어 프레젠테이션
4. 시간을 절약한다
 - 많은 양의 말로 설명하는 대신 간단한 설명과 함께 이해를 돕는 자료 이용

메시지를 구성할 때 주의해야할 점
프레젠테이션의 전체 메시지를 구성할 때는, 단순성 이해도, 흥미도, 정확성을 고려해야한다
단순성 : 한 장의 슬라이드에 하나의 메시지만 담길 때 청중들의 시선이 발표자에게 집중된다
이해성 : 프레젠테이션의 목적(전달, 설득, 동기
흥미성
정확성

3. 비주얼 프레젠테이션 7단계
1단계 3P 분석 : 듣는 사람의 수, 장소에 따라 시각정보 도구를 다르게 구성한다
2단계 시각정보의 도구 선택
3단계 시각정보 제작 부분 결정
4단계 이야기 구조 그리기
5단계 시각정보 제작
6단계 유인물 제작
7단계 예행 연습

3p란
목적(purpose)
-프레젠테이션을 하는 이유 분석
-프레젠테이션을 통해 얻고자 하는 것
(설득,정보전달, 행동의 변화, 동기부여 등)
청중(people)
-프레젠테이션에 대한 청중의 기대치 분석
-참여하는 사람들의 인원 수, 핵심 인물(key man)
-성별, 사람들의 관심사 등
장소(place)
-프레젠테이션 진행 장소의 환경 분석
-좌석 배치, 창문의 유무, 밝기
-빔 작동 여부, 마이크 유무, 음향기기 체크
