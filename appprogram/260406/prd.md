# PRD: 상식 퀴즈 게임

## 1. 개요

### 1.1 제품 목적
사용자가 한국사, 과학, 지리, 일반상식 분야의 지식을 테스트하고 학습할 수 있는 4지선다 객관식 퀴즈 게임.
난이도 선택, 즉각적인 오답 분석 피드백, 콤보 보너스 점수, 결과 공유 기능을 통해 학습과 경쟁 동기를 동시에 제공한다.

### 1.2 목표 사용자
- 지식 향상에 관심 있는 일반 사용자
- 퀴즈를 통해 학습하고 싶은 학생 및 성인

### 1.3 핵심 가치
- 난이도 선택으로 누구나 접근 가능한 학습 경험
- 즉각적인 오답 분석으로 단순 정오 확인을 넘어선 학습 효과
- 콤보 시스템과 타이머로 긴장감 있는 게임 경험
- 결과 공유를 통한 소셜 동기 부여

---

## 2. 게임 모드

### 2.1 모드 구성
카테고리와 난이도를 각각 선택하는 방식으로 진행한다.

| 선택 항목 | 옵션 |
|---------|------|
| 카테고리 | 한국사 / 과학 / 지리 / 일반상식 |
| 난이도 | 쉬움 / 보통 / 어려움 |

- 카테고리 선택 후 난이도를 선택하면 해당 조합의 문제 풀에서 10문제 출제
- 문제 및 보기 순서는 매 게임마다 랜덤 셔플

### 2.2 게임 플로우

```
시작 화면 (닉네임 입력)
    ↓
카테고리 선택
    ↓
난이도 선택 (쉬움 / 보통 / 어려움)
    ↓
문제 출제 (4지선다 + 타이머)
    ↓
답변 선택 또는 시간 초과
    ↓
즉시 피드백 (정답/오답 + 오답 분석 해설)
    ↓
다음 문제
    ↓
최종 결과 화면 (점수, 콤보, 등급)
    ↓
결과 저장 및 공유
```

## 3. 카테고리 및 문제 구성

### 3.1 카테고리

| 카테고리 | 설명 |
|---------|------|
| 한국사 | 고대~현대 한국 역사, 인물, 사건 |
| 과학 | 물리, 화학, 생물, 지구과학 기초 개념 |
| 지리 | 세계 지리, 국가, 수도, 지형, 기후 |
| 일반상식 | 문화, 사회, 스포츠, 시사, 언어 |

### 3.2 문제 풀 구성
카테고리(4) × 난이도(3) = **12개 JSON 파일**, 각 파일당 최소 10문제

| | 쉬움 | 보통 | 어려움 |
|---|---|---|---|
| 한국사 | 10문제 | 10문제 | 10문제 |
| 과학 | 10문제 | 10문제 | 10문제 |
| 지리 | 10문제 | 10문제 | 10문제 |
| 일반상식 | 10문제 | 10문제 | 10문제 |
| **합계** | | | **최소 120문제** |

게임 1회 진행 시 해당 카테고리+난이도 파일에서 **랜덤 10문제** 출제

## 4. 타이머 시스템

### 4.1 난이도별 제한시간

| 난이도 | 문제당 제한시간 |
|------|------------|
| 쉬움 | 30초 |
| 보통 | 20초 |
| 어려움 | 15초 |

### 4.2 타이머 동작
- 문제 표시와 동시에 타이머 시작
- 화면 상단에 프로그레스 바로 남은 시간 시각화
- 남은 시간 5초 이하 시 타이머 바 색상 변경 (회색 → 빨강)
- 시간 초과 시 자동 오답 처리 후 피드백 화면으로 이동

### 4.3 타이머 보너스 점수
남은 시간 비율에 따라 보너스 점수 추가

| 남은 시간 비율 | 보너스 점수 |
|------------|-----------|
| 70% 이상 | +5점 |
| 40% ~ 69% | +3점 |
| 10% ~ 39% | +1점 |
| 10% 미만 또는 시간초과 | +0점 |

## 5. 점수 및 등급 체계

### 5.1 기본 점수

| 난이도 | 정답 기본 점수 | 타이머 보너스 최대 |
|------|------------|--------------|
| 쉬움 | 5점 | +5점 |
| 보통 | 10점 | +5점 |
| 어려움 | 15점 | +5점 |

### 5.2 콤보 보너스
연속 정답 시 추가 점수 지급

| 연속 정답 수 | 콤보 보너스 |
|-----------|-----------|
| 3연속 | +5점 |
| 5연속 | +10점 |
| 10연속 | +20점 |

- 오답 또는 시간 초과 시 콤보 초기화
- 콤보 달성 시 화면에 "3 COMBO!", "5 COMBO!" 텍스트 팝업 표시

### 5.3 최고 점수 계산 예시 (어려움, 10문제 전정답)
- 기본: 15점 × 10문제 = 150점
- 타이머 보너스: 5점 × 10문제 = 50점
- 10연속 콤보: +20점
- **최고 합계: 220점**

### 5.4 등급 기준
정답률(정답 수 / 전체 문제 수) 기준으로 등급 산정

| 등급 | 정답률 |
|-----|------|
| S | 90% 이상 |
| A | 75% ~ 89% |
| B | 60% ~ 74% |
| C | 40% ~ 59% |
| D | 40% 미만 |

## 6. 피드백 및 해설 방식

### 6.1 정답 시
- 선택한 보기 초록색 하이라이트
- "정답입니다!" 메시지 표시
- 획득 점수 + 타이머 보너스 점수 표시
- 콤보 달성 시 콤보 텍스트 추가 표시
- 1~2줄 핵심 해설 텍스트

### 6.2 오답 시 (오답 분석 포함)
- 선택한 보기 빨간색 하이라이트
- 정답 보기 초록색 하이라이트
- "오답입니다!" 메시지 표시
- 핵심 해설 텍스트
- **각 보기별 오답 분석**: 내가 선택한 보기가 왜 틀렸는지 간략히 설명

**오답 분석 예시:**
```
문제: 고려를 건국한 인물은?

① 왕건  ✓ 정답 — 918년 고려 건국
② 이성계  ✗ — 이성계는 1392년 조선을 건국한 인물
③ 장보고  ✗ — 장보고는 통일신라 시대 해상 무역 인물
④ 견훤  ✗ — 견훤은 후백제를 건국한 인물
```

### 6.3 공통
- 시간 초과 시 오답과 동일한 피드백 + "시간 초과!" 메시지 추가
- "다음 문제" 버튼으로 수동 진행 (자동 진행 없음)

## 7. 리더보드 및 순위 시스템

### 7.1 저장 방식
- LocalStorage 기반 로컬 저장
- 카테고리 + 난이도 조합별 독립 순위 관리
- 저장 항목: 닉네임, 점수, 등급, 정답률, 최고 콤보, 카테고리, 난이도, 날짜

### 7.2 리더보드 구성
- 카테고리별 + 난이도별 필터 탭
- 각 조합 상위 10명 표시
- 내 기록 하이라이트 표시
- 표시 컬럼: 순위 / 닉네임 / 점수 / 등급 / 정답률 / 최고콤보 / 날짜

### 7.3 결과 공유 기능
게임 종료 후 결과 화면에서 "결과 공유" 버튼 제공

**공유 텍스트 자동 생성 예시:**
```
📝 상식 퀴즈 결과
카테고리: 한국사 | 난이도: 어려움
점수: 185점 | 등급: A
정답률: 8/10 (80%) | 최고 콤보: 5연속
#상식퀴즈 #한국사
```

- 클립보드 복사 버튼
- (선택) 결과 카드 이미지로 저장 (canvas 활용)

## 8. 화면 구성 (UI)

### 8.1 디자인 원칙 (미니멀형)
- 흰 배경, 깔끔한 텍스트 중심
- 불필요한 애니메이션 배제, 기능 중심 레이아웃
- 폰트: 시스템 기본 폰트 또는 Noto Sans KR
- 컬러: 기본 회색 계열 + 정답(#22c55e 초록) / 오답(#ef4444 빨강) / 강조(#3b82f6 파랑)

### 8.2 화면 목록

| 화면 | 설명 |
|-----|------|
| 시작 화면 | 닉네임 입력, 카테고리 선택 |
| 난이도 선택 화면 | 쉬움/보통/어려움 선택, 난이도 설명 |
| 퀴즈 화면 | 문제 + 보기 4개 + 타이머 바 + 현재 점수/콤보 |
| 피드백 화면 | 정오답 표시 + 오답 분석 해설 |
| 결과 화면 | 최종 점수, 등급, 정답률, 콤보, 저장/공유 버튼 |
| 리더보드 화면 | 카테고리×난이도 필터 + 상위 10명 목록 |

## 10. 기술 스택

| 항목 | 기술 |
|------|------|
| 언어 | HTML5, CSS3, JavaScript (Vanilla ES6+) |
| 데이터 | JSON 파일 (12개 문제 파일) |
| 저장소 | LocalStorage (순위 기록) |
| 빌드 도구 | 없음 (정적 파일, 브라우저에서 바로 실행) |
| 외부 라이브러리 | 없음 |

### 10.1 파일 구조
```
/
├── index.html
├── style.css
├── js/
│   ├── main.js          # 앱 진입점, 화면 전환
│   ├── quiz.js          # 퀴즈 로직 (문제 출제, 채점)
│   ├── timer.js         # 타이머 관리
│   ├── score.js         # 점수, 콤보, 등급 계산
│   ├── leaderboard.js   # 순위 저장/조회
│   └── share.js         # 결과 공유 텍스트 생성
└── questions/
    ├── korean_history_easy.json
    ├── ... (총 12개)
    └── general_hard.json
```

## 12. 개발 우선순위

| 우선순위 | 기능 |
|---------|------|
| P0 (필수) | 문제 출제, 4지선다 선택, 타이머, 즉시 피드백 + 오답 분석, 점수 계산 |
